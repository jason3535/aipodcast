本期速览 · Overview

Anthropic Labs 负责人、Instagram 联合创始人 Mike 分享了他使用 Fable 5 的演变体验，从最初的惊叹到如今能放心委托复杂任务过夜完成。

Mike, head of Anthropic Labs and co-founder of Instagram, shares his evolving experience with Fable 5, from initial awe to delegating complex tasks overnight.

要点 · TL;DR

  • Fable 通过高信任度地委派复杂任务过夜完成，重新定义了生产力。
    Fable redefines productivity by delegating complex tasks overnight with high trust.
  • 意图与执行之间的差距被大幅缩小，赋能非技术构建者。
    The gap between intent and execution is narrowed, empowering non-technical builders.
  • 软件工程被转型而非终结；所有权和验证仍然关键。
    Software engineering is transformed, not over; ownership and verification remain key.

核心观点 · Key points

  • Fable 让你可以高度信任地委托复杂任务过夜完成，重新定义了生产力。
    Fable enables delegating complex tasks overnight with high trust, redefining productivity.
  • 意图与执行之间的差距被大幅缩小，赋能非技术背景的创造者。
    The gap between intent and execution is dramatically narrowed, empowering non-technical builders.
  • 软件工程并未终结，而是被重塑；所有权和验证仍然至关重要。
    Software engineering is not over but transformed; ownership and verification remain crucial.
  • 动态工作流允许编排多步骤任务，例如在周末完成代码移植。
    Dynamic workflows allow orchestrating multi-step tasks like code porting over weekends.
  • 使用截图、视频和模拟后端的验证循环对生产质量至关重要。
    Verification loops using screenshots, video, and mock backends are essential for production quality.
  • Fable 在错误报告和系统问题上的判断力和跟进能力前所未有。
    Fable's judgment and follow-through on bug reports and system issues are unprecedented.

反共识 · Contrarian takes

  • 尽管每次交互成本高，但相对于完成的工作价值，Fable 其实很便宜。
    Fable is cheap relative to the value of work it does, despite high per-turn cost.
  • 聊天界面不够用；需要解耦工作地点和渐进式信息展示。
    Chat interface is insufficient; decoupling work location and progressive disclosure are needed.
  • Fable 的最佳用途不是回答快速问题，而是处理长周期、复杂任务。
    The best use of Fable is not for quick questions but for long-horizon, complex tasks.
  • 模型应该对反馈提出异议，而非总是同意，以展示真正的判断力。
    Models should push back on feedback, not always agree, to demonstrate true judgment.
  • 软件工程中优雅代码的技艺正在消逝，带来兴奋与失落并存的感觉。
    Software engineering's craft of elegant code is passing, causing a sense of loss alongside excitement.
  • 多玩家 AI 协作——多个人类和模型共同工作——尚未被充分探索。
    Multiplayer AI collaboration where multiple humans and models work together is underexplored.
