本期速览 · Overview

Stripe 设计负责人 Chris Pucket 分享他如何利用 AI 工具构建自己的第一个 iOS 应用，实现了 20 年的梦想，并探讨了好奇心和灵魂在设计中的重要性。

Chris Pucket, design leader at Stripe, shares how he built his first iOS app using AI tools, fulfilling a 20-year dream, and discusses the importance of curiosity and soul in design.

  • 像 Claude Code 这样的 AI 工具让非程序员无需许可就能构建完整的 iOS 应用。
    AI tools like Claude Code let non-coders build full iOS apps without permission.

核心观点 · Key points

  • AI工具如Claude Code让非程序员无需许可即可构建完整iOS应用。
    AI tools like Claude Code enable non-coders to build full iOS apps without permission.
  • 说“我不知道”是一种超能力，能邀请与AI进行更深入的合作。
    Saying 'I don't know' is a superpower that invites deeper collaboration with AI.
  • Claude Code中的技能是可重复的指令，能减轻脑力负担并加速工作流。
    Skills in Claude Code are repeatable instructions that offload mental load and speed up workflows.
  • AI原生设计师必须展示好奇心和动手构建能力，才能在招聘中脱颖而出。
    AI native designers must demonstrate curiosity and hands-on building to stand out in hiring.
  • 随着AI使质量和工艺无处不在，灵魂和个人表达将成为下一个差异化因素。
    Soul and personal expression will become the next differentiator as AI makes quality and craft ubiquitous.

反共识 · Contrarian takes

  • LLM不擅长创意探索，但若正确引导，在发散思维方面表现出色。
    LLMs are poor at creative exploration but excel at divergent thinking when taught properly.
  • MCP并未消亡，但由于上下文窗口限制，将被更好的语义API和CLI取代。
    MCPs are not dead but will be replaced by better semantic APIs and CLIs due to context window limits.
  • 用AI做越来越多的事可能导致贫瘠；应专注于真实且自豪的输出。
    Using AI to do more and more can lead to barrenness; focus on authentic, proud output instead.
  • 基本的Squarespace作品集是危险信号；候选人应使用AI创建定制化、个性化的网站。
    A basic Squarespace portfolio is a red flag; candidates should use AI to create custom, personal sites.
  • 自主运行的智能体有风险；人在回路中的审查对质量和真实性至关重要。
    Agents running autonomously are risky; human-in-the-loop review is essential for quality and authenticity.
